## This Repo

This repository contains the Syri AI marketing website — a modern, responsive landing page showcasing our capabilities.

### Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom design system with animations, glassmorphism, responsive layout
- **JavaScript** — Interactive features (cursor glow, scroll animations, mobile nav)
- **Google Fonts** — Space Grotesk, Inter
- **GitHub Pages** — Automated deployment via GitHub Actions

### Run Locally

```bash
git clone https://github.com/easgnu/Syri_AI.git
cd Syri_AI
open index.html
```

### Deploy

Pushes to `main` auto-deploy to GitHub Pages via the included workflow.
